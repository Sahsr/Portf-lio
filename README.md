# 🌌 Sobre o Projeto

Meu portfólio possui um design futurista inspirado em tecnologia, neon e universo cyberpunk/NASA, com animações suaves, efeitos glow e layout responsivo.

O objetivo deste projeto é mostrar minha evolução como desenvolvedora Full Stack em formação, com foco em:

- 💻 Back-End
- 🎨 Front-End
- 🗄️ Banco de Dados
- 🚀 Desenvolvimento Web
